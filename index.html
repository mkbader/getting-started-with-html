What is the WWW?

Major Pieces

Internet

What is HTML?
Text or content, markup (what it looks like), references to other
documents, and links to other pages

Deeper dive into Markup
Tags- change the way content is displayed
Attributes- propoerties of some tags that
specify more specific display behaviors

Includes b, em (emphasis), a, image, for example,
placing the word word in bold tags looks like this word
There are void tags that do not need to be closed, e.g, img
Together the tags and the content are called an element
Tags can be inline or block, the difference is block creates an invisible
box around the element that can have it's own attributes such as height and width
HTML Documents are composed of various sections
Doctype, head (typically titles, metadata, and any css file or javascript
would be exposed here, and the body. In this course, we will mainly focus on body.
